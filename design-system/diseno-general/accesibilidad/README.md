# Accesibilidad ERP

## Reglas base
- Contraste mínimo WCAG AA para texto y controles.
- Foco visible y consistente en todos los interactivos.
- Navegación completa por teclado.

## Formularios
- Etiquetas siempre asociadas a inputs.
- Mensajes de error cercanos y descriptivos.
- No usar solo color para indicar error o éxito.

## Tablas y datos
- Cabeceras semánticas en tablas.
- Orden y estado anunciables para lectores de pantalla.
- Objetivos táctiles de al menos 44x44px en mobile.
