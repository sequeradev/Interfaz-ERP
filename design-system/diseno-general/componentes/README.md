# Componentes ERP

## Navegación
- Sidebar con estado activo claro.
- Breadcrumbs en vistas de detalle.
- Tabs para submódulos de una misma entidad.

## Datos
- Tabla con filtros, orden y paginación.
- Tarjetas KPI para métricas resumidas.
- Badges de estado (activo, pendiente, bloqueado).

## Interacción
- Formularios por secciones con validación inline.
- Modales para acciones críticas.
- Toasts para confirmación no bloqueante.
