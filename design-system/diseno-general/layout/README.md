# Layout ERP

## Estructura base
- Sidebar izquierda fija con módulos principales.
- Header superior con búsqueda global, alertas y perfil.
- Contenedor central adaptable por tipo de vista.

## Rejilla
- Desktop: grilla de 12 columnas.
- Tablet: grilla de 8 columnas.
- Mobile: grilla de 4 columnas.

## Espaciado
- Unidad base: 8px.
- Separación entre bloques: 16px / 24px / 32px según jerarquía.
- Margen lateral estándar: 24px en desktop, 16px en mobile.
