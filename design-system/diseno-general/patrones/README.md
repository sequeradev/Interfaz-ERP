# Patrones ERP

## Listado + detalle
- Panel principal con tabla.
- Apertura de detalle en vista completa o panel lateral.
- Acciones rápidas por fila (ver, editar, duplicar).

## Flujo de creación/edición
- Botón primario fijo en contexto.
- Formulario dividido por bloques funcionales.
- Guardado con feedback inmediato (éxito/error).

## Estados de pantalla
- Loading con skeletons.
- Vacío con mensaje y CTA principal.
- Error con descripción y acción de recuperación.
